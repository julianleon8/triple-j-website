# Research Vault — Triple J Metal LLC

Drop research output here so Claude can read it across sessions without re-running searches.

## Folder Structure

| Folder | What goes here |
|---|---|
| `competitors/` | NotebookLM competitor intel, manual research, Google SERP notes |
| `keywords/` | Keyword lists, search volume data, ranking reports |
| `markets/` | City-level notes — local demand signals, Google Trends, demographics |
| `inspiration/` | Screenshots and notes on sites/designs you like |
| `customers/` | Patterns from real customer calls, common questions, objections |

## How to Use

1. Run your NotebookLM prompt
2. Copy the output
3. Save it as a `.md` file in the right subfolder (e.g. `competitors/central-texas-builders.md`)
4. Tell Claude: "I dropped new research in `research/competitors/`"
5. Claude reads it and uses it immediately — no re-pasting needed

## Naming Convention

```
competitors/[company-name].md
competitors/full-analysis-[date].md
keywords/harker-heights-keywords.md
markets/harker-heights.md
inspiration/[site-or-feature-name].md
```
