- [ ] Set service areas: Waco, Austin, Dallas, Temple, Killeen, Round Rock, Georgetown

- [ ] Add all services with descriptions
- [ ] Upload 10+ photos (completed jobs, team, materials)
- [ ] Add business hours
- [ ] Add website + phone number
- [ ] Seed 5 Q&A entries (common questions)
- [ ] Add booking/quote link
